How to integrate (notes)

  • Spline: Publish your scene and embed via <iframe> or use the Spline Runtime in React for more control and per-key interactions.
  • GSAP: Use the CDN for prototypes (https://cdnjs.cloudflare.com/ajax/libs/gsap/3.x/gsap.min.js) or npm for production. Great for timeline and scroll-triggered reveal animations.
  • Framer Motion: Used inside React components (Next.js). Replace the HTML-based hover animations with motion components for richer interactions.
  • Particles: This page includes a lightweight canvas particle system (below) which you can tune or swap for tsParticles.
